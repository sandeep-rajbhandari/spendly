❯ how to loop in python? 

⏺ This project uses Python with Flask as the web framework. The tech stack
  includes:
  - Backend: Python with Flask
  - Testing: pytest, pytest-flask
  - Frontend: HTML templates (in the templates directory)

  Now, regarding your question about looping in Python, here are the main ways
  to loop:

  1. for loop - Used to iterate over sequences (lists, tuples, strings) or
  ranges:
  # Loop through a list
  for item in [1, 2, 3]:
      print(item)

  # Loop through a range
  for i in range(5):  # 0 to 4
      print(i)

  # Loop through a string
  for char in "hello":
      print(char)

  2. while loop - Used to repeat code while a condition is true:
  count = 0
  while count < 5:
      print(count)
      count += 1

  3. Looping through dictionaries:
  my_dict = {"a": 1, "b": 2, "c": 3}

  # Loop through keys
  for key in my_dict:
      print(key)

  # Loop through values
  for value in my_dict.values():
      print(value)

  # Loop through key-value pairs
  for key, value in my_dict.items():
      print(key, value)

  4. Loop control statements:
  # break - exit the loop
  for i in range(10):
      if i == 5:
          break
      print(i)

  # continue - skip to next iteration
  for i in range(5):
      if i == 2:
          continue
      print(i)

  # else - runs after loop completes normally (without break)
  for i in range(3):
      print(i)
  else:
      print("Loop completed")
